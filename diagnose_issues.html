🔍 Page Builder Diagnostics

This page will help diagnose issues with the page builder.

Quick Fixes

If Backend Connection Failed:

cd backend
python server.py

If Frontend Not Working:

cd frontend
npm run dev

If Text Component Not Showing:

  1. Make sure backend is running (see above)
  2. Hard refresh the page builder (Ctrl+Shift+F5)
  3. Check browser console for errors (F12)
  4. Try clicking on "Text Block" in the left sidebar

If Text Align Not Working:

  1. Select the text/heading component
  2. Open Typography panel in right sidebar
  3. Click the center align button
  4. Click "Save" button in top bar
  5. Click "Publish" button in top bar
  6. Hard refresh the live page (Ctrl+Shift+F5)
