## Key Features

### Live Preview
- All changes update immediately on canvas
- No need to save to see changes
- Real-time visual feedback

- Typography only shows for text elements
- Container controls only for sections/flexbox
- Smart UI that adapts to component type

### Professional Controls
- Sliders with numeric inputs
- Visual box model for spacing
- Color pickers with presets
- Image upload with preview
- Icon-based button groups

- Delete: Remove selected component

- Ctrl+D: Duplicate component
- Backspace: Safe for text editing (doesn't delete)
